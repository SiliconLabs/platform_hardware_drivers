typedef void(* vcnl4040_norm_interrupt_callback_t) (vcnl4040_irq_source_t irq)

VCNL4040 interrupt callback function; normal operation.

This callback function is executed from interrupt context when the user has enabled one of the VCNL4040 interrupt sources. This callback acts differently based on the driver configuration. When driver is operating in normal mode (ALS/PS upper/lower threshold interrupts), this function sets a driver flag that should be checked in the main application process action sequence. When driver is configured for proximity logic interrupt, driver flag is used to indicate object proximity - the INT pin is pulled low when an object is close to the sensor (value is above high threshold) and is pulled high when the object moves away (value is below low threshold).
