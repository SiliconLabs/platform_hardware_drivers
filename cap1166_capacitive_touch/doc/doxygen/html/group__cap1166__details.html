◆ cap1166_detect_touch()

sl_status_t cap1166_detect_touch ( cap1166_handle_t *  cap1166_handle,
uint8_t *  in_sens 
)

Detects touch on sensor inputs to determine which one is detected or released.

Parameters
[in]cap1166_handleThe cap1166_handle_t instance to use.
[out]in_sensResults of detecting touch on sensor inputs. When in_sens[index] = 2 - released detection in_sens[index] = 1 - touched detection in_sens[index] = 0 - no touch detection index which means the sensor inputs touch in range CS1 to CS6.
Note
    Function detects touch on sensor inputs and checks if touch is detected
    or if touch is released.
    If the noise flag was set corresponding to sensor input channel,
    the touch will not be detected.
Returns
SL_STATUS_NULL_POINTER Pointer to NULL
Return values
SL_STATUS_OKSuccess
SL_STATUS_TRANSMITSPI transmission error

Detects touch on sensor inputs to determine which input touch is detected or released.
