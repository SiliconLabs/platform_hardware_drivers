Detailed Description

Capacitive touch sensor driver.

Driver for the Silicon Labs Cap1166 Capacitive Touch Sensor.


cap1166 example code

Basic example for touch sensor: 

#include "cap1166.h"

int main( void )
{

  ...

  cap1166_init(...);
  cap1166_config(...);

  ...

  cap1166_detect_touch(..., &out);

  ...

}
