◆ ak9753_deinit()

sl_status_t ak9753_deinit ( void  )

De-initalize the AK9753 sensor.

This function returns the GPIO pin used for INT1 to disabled state, and powers down the device if PDN is present.

Note
The purpose of de-initialization is to shutdown the AK9753 as part of a complete shutdown sequence for an EFM32/EFR32-based system.
Returns
  • SL_STATUS_OK if there are no errors.
  • SL_STATUS_TRANSMIT if I2C transmit failed.
  • SL_INVALID_PARAMETER if invalid parameter.

This function is called in the main application process action sequence with normal mode operation.

The function checks the driver interrupt flag and if set, reads the sensor interrupt flag register and returns interrupt source status.
