 Initiates a measurement cycle and reads temp and RH data from the SHTC3. SHTC3 power mode, data return order, I2C clock stretching, and data CRC checking are all selectable. This function receives raw data over I2C as well as calculates temperature (deg C) and RH (%) from the raw data. Raw and calculated values are stored in the data structure. This function is a non-blocking function, so instead of waiting for the sensor to finish reading the temperature as well as humidity, this function will return SL_STATUS_BUSY when the sensor has not finished measuring.
